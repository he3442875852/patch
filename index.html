FAQ

Questions Buyers Usually Ask First.

What do I need to send for a quote?

Send artwork, size, quantity, patch type if known, backing preference, needed date and shipping country.

Can you recommend a material?

Yes. We can review detail level, small text, color separation, use case and backing needs before recommending a material.

Can I approve a proof before production?

Yes. Proof approval is part of production preparation so shape, color, border and backing can be confirmed.

What file types can I upload?

JPG, JPEG, PNG, PDF, AI, SVG and EPS are accepted by the quote form, up to 8MB.
